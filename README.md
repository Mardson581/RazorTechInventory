
# RazorTechInventory

Aplicação web simples para gerenciamento de inventário de TI, desenvolvida com ASP.NET Core Razor Pages.

O projeto tem como objetivo demonstrar, de forma prática, o uso de Razor Pages, organização em camadas e boas práticas iniciais de backend, servindo como MVP para portfólio.




## Stack utilizada

ASP.NET Core 8

Razor Pages

Entity Framework Core

InMemory Database

C#


## Funcionalidades

- CRUD de marcas

- CRUD de modelos de dispositivos

- CRUD de dispositivos

- Relacionamentos entre entidades

- Validações com Data Annotations
